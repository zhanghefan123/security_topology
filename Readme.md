# 2. gvm 用来进行 go 版本管理

- 安装命令：

```shell
bash < <(curl -s -S -L https://raw.githubusercontent.com/moovweb/gvm/master/binscripts/gvm-installer)
```

- 使用命令：
  - gvm install go1.4
  - gvm use go1.4
  - 如果卡住了可以更换为 gvm install go1.4 --source=https://github.com/golang/go
